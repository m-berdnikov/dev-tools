### Windows

Multiple selection by search

```
Shift + Ctrl + L
```

Open Terminal

```
Ctrl + `
```

Duplicate Line

```
Ctrl + Shift + D
```

### Mac OS

Cвернуть/развернуть блок кода

```
Shift + клик по раскрывающей/закрывающей стрелке -
```

Open/close left bar

```
CMD + B
```

Open/close Terminal

```
Control + `
```

Close all tabs

```
CMD + K W
```

Open Extensions

```
Shift + CMD + X
```

Open Explorer

```
Shift + CMD + E
```

Open Source Control

```
Shift + Control + G -
```

Open Search

```
Shift + CMD + F
```

Show bottom pannel (with terminal, output, etc.)

```
CMD + J
```

Split to right

```
CMD + \
```
